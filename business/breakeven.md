# Breakeven Analysis

## Cost per Active User (CPAU)
- **Compute**: $0.10/user/month (assuming 100 prompt evaluations/user/month at $0.001 per evaluation)
- **Storage**: $0.05/user/month (1GB storage/user at $0.05/GB/month)
- **Bandwidth**: $0.05/user/month (10GB bandwidth/user/month at $0.005/GB)
- **Total CPAU**: $0.20/user/month

## Pricing Tiers
1. **Basic**: $20/month
   - Features: 100 prompt evaluations/month, 1GB storage, basic analytics
2. **Pro**: $50/month
   - Features: 500 prompt evaluations/month, 5GB storage, advanced analytics, API access
3. **Enterprise**: $150/month
   - Features: Unlimited prompt evaluations, 20GB storage, premium analytics, dedicated support, custom integrations

## Customer Acquisition Cost (CAC) Range
- **Lower Bound**: $50/user (organic growth, content marketing, SEO)
- **Upper Bound**: $200/user (paid advertising, influencer marketing, targeted outreach)
- **Average CAC**: $125/user

## Lifetime Value (LTV) Estimate
- **Average Customer Lifespan**: 12 months
- **Average Revenue per User (ARPU)**: $70/month (weighted average of tiers)
- **LTV**: $70/month × 12 months = $840/user

## Break-even Users Count
- **Monthly Costs**: $5,000 (hosting, salaries, marketing, overhead)
- **Break-even Users**: $5,000 / ($70 - $0.20) = 72 users

## Path to $10K MRR
- **Basic Tier**: 300 users × $20/month = $6,000 MRR
- **Pro Tier**: 100 users × $50/month = $5,000 MRR
- **Enterprise Tier**: 20 users × $150/month = $3,000 MRR
- **Total**: 420 users generating $14,000 MRR
