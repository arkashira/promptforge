# LLM Vendor Integration

This project provides a simple API for integrating new LLM vendors and testing them.

## Usage

1. Create an instance of `LLMIntegration`.
2. Add a vendor using `add_vendor`.
3. Test a vendor using `test_vendor`.
4. List all vendors using `list_vendors`.
5. Serialize the integration to JSON using `to_json`.
6. Deserialize the integration from JSON using `from_json`.
