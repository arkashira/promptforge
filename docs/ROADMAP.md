# ROADMAP.md

## Roadmap

### MVP Milestone (Must-Have for Launch)

The Minimum Viable Product (MVP) will provide a foundational platform for testing and evaluation of Large Language Models. The following features are critical for the MVP:

#### 1. **Model Integration**

* Integrate with popular LLM frameworks (e.g., vLLM, SGLang) via APIs or SDKs
* Support for multiple model architectures and sizes
* MVP-critical: **vLLM integration**

#### 2. **Prompt Library**

* Develop a comprehensive library of prompts for testing and evaluation
* Include a variety of prompt types (e.g., instructions, edge cases, domain-specific)
* MVP-critical: **initial prompt library with 100+ prompts**

#### 3. **Evaluation Metrics**

* Implement a set of evaluation metrics for LLM performance (e.g., accuracy, fluency, relevance)
* Support for custom metric definitions and calculations
* MVP-critical: **basic accuracy and fluency metrics**

#### 4. **User Interface**

* Develop a user-friendly web interface for model testing and evaluation
* Support for model selection, prompt selection, and evaluation metric configuration
* MVP-critical: **basic web interface with model and prompt selection**

#### 5. **Data Storage and Management**

* Implement a data storage system for model performance data and evaluation metrics
* Support for data filtering, sorting, and export
* MVP-critical: **basic data storage and filtering**

### v1 Phase: **Streamlining Development and Research**

The v1 phase will focus on enhancing the platform's capabilities to streamline development and research processes.

#### 1. **Advanced Model Integration**

* Support for more LLM frameworks and architectures
* Integration with popular research libraries (e.g., Hugging Face Transformers)
* v1: **add support for 2-3 additional LLM frameworks**

#### 2. **Prompt Generation and Editing**

* Develop a prompt generation tool for creating custom prompts
* Implement a prompt editing feature for refining and optimizing prompts
* v1: **basic prompt generation and editing**

#### 3. **Collaboration and Sharing**

* Introduce a collaboration feature for multiple users to work on prompts and models together
* Support for sharing prompts and models with others
* v1: **basic collaboration and sharing**

### v2 Phase: **Advanced Research and Development**

The v2 phase will focus on advanced research and development capabilities.

#### 1. **Explainability and Interpretability**

* Implement techniques for explaining and interpreting LLM decisions
* Support for visualizing model performance and behavior
* v2: **basic explainability and interpretability**

#### 2. **Active Learning and Human-in-the-Loop**

* Develop an active learning system for improving model performance
* Support for human-in-the-loop feedback and correction
* v2: **basic active learning and human-in-the-loop**

#### 3. **Integration with External Tools and Services**

* Integrate with popular research tools and services (e.g., GitHub, Jupyter Notebooks)
* Support for seamless integration with external workflows
* v2: **basic integration with 2-3 external tools and services**
